### Bool

Для введения типа Bool понадобятся литералы булевого типа $\texttt{True}$ и $\texttt{False}$, а также следующее изменение уравнений (остальные оставим неизменными):

| Конструкция | Уравнение |
|-----------|-----------------|
| $\texttt{True}$ |  $\llbracket \texttt{True} \rrbracket = \text{bool}$ |
| $\texttt{False}$ | $\llbracket \texttt{False} \rrbracket = \text{bool}$ |
| $E_1 \mathbin{==} E_2$ | $\llbracket E_1 \rrbracket = \llbracket E_2 \rrbracket$, $\llbracket E_1 \mathbin{==} E_2 \rrbracket = \text{bool}$ |
| $E_1 \mathbin{>} E_2$ | $\llbracket E_1 \rrbracket = \llbracket E_2 \rrbracket = \text{int}$, $\llbracket E_1 \mathbin{>} E_2 \rrbracket = \text{bool}$ |
| $\texttt{if}\ (E)\ S$ | $\llbracket E \rrbracket = \text{bool}$ |
| $\texttt{while}\ (E)\ S$ | $\llbracket E \rrbracket = \text{bool}$ |

Программы вида $\texttt{x = a == b + c == d}$ теперь отвергаются, что увеличило число false positive, значит точность упала.

Число true positive в целом не изменилось, так как по старой семантике int (0/1) неявно использовался как $\texttt{bool}$, поэтому полнота осталась неизменной.

### Array

Вводим тип $\text{array}(T)$ для массивов с элементами типа $T$. Новые уравнения:

| Конструкция | Уравнение |
|-----------|-----------------|
| $\{E_1, \ldots, E_n\}$ | $\llbracket E_1 \rrbracket = \cdots = \llbracket E_n \rrbracket$, $\llbracket \{E_1,\ldots,E_n\} \rrbracket = \text{array}(\llbracket E_1 \rrbracket)$ |
| $\{\}$ | $\llbracket \{\} \rrbracket = \text{array}(T)$ (свежий $T$) |
| $E_1[E_2]$ | $\llbracket E_1 \rrbracket = \text{array}(\llbracket E_1[E_2] \rrbracket)$, $\llbracket E_2 \rrbracket = \text{int}$ |
| $E_1[E_2] = E_3$ | $\llbracket E_1 \rrbracket = \text{array}(\llbracket E_3 \rrbracket)$, $\llbracket E_2 \rrbracket = \text{int}$ |

**Пример вывода типов для программы:**

```
main() {
    var x,y,z,t;
    x = {2,4,8,16,32,64}; \\ [[x]] = array(int)
    y = x[x[3]];          \\ [[y]] = int
    z = {{},x};           \\ [[z]] = array(array(int))
    t = z[1];             \\ [[t]] = array(int)
    t[2] = y;
}
```
