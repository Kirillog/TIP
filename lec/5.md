## Анализ размера переменных

### Решетка размеров

```
                any
                |
           bigint [-inf, inf]
                |
     --int [-MININT, MAXINT]--
     |                       |  
byte [-128, 127]    char [0, 65535]
     |                       |
     -------------------------
                |
            bool [0, 1]
                |
                ⊥
```

Для анализа зададим решетку отображений из нод графа потока управления в решетку размеров,
получим обычный forward flow-sensitive may анализ.


### Правила вычисления

Константу $n$ вычисляем в наименьший размер, который его содержит:
$$
eval(\sigma, 100) = \texttt{byte} \\
eval(\sigma, 1000) = \texttt{char}
$$

$$
eval(\sigma, x) = \sigma(x)
$$

Вычисление операции аналогично интервальному анализу, выбираем наименьший интервал, который содержит результат операции $\overline{op}$:
$$
eval(\sigma,E_1\,op\,E_2) =
  eval(\sigma, E_1)\,\overline{op}\,eval(\sigma, E_2)
$$

$$
eval(\sigma, \mathtt{input}) = \texttt{any}
$$

$$
eval(\sigma, (T)\,E) = T
$$

### Пример программы и результат

```tip
main() {
    var flag, b, s, c, x, y, z, result;
    b    = 1;       // bool
    s    = 100;     // byte
    c    = 1000;    // char
    x    = s + s;   // int
    flag = x > 200; // bool
    y    = flag * c + s; // int
    z    = input;   // any
    result = z + x; // any
    return result;
}
```
