
### Программа

```tip
if (a) {
    if (b) {
        open();
    }
}
...
if (a) {
    if (b) {
        close();
    }
}
```

Доказать ее корректность не получится, если взять
$\text{Path} = \{a=0,\ a\ne 0, b=0, b\neq0\}$. Из такого набора предикатов мы не можем получить предикат $(a\ne 0 \wedge b\ne 0)$ простым способом.

В качестве решения либо группировать переменные вокруг операций и рассмотреть все предикаты с их использованием, либо упомянутый в лекции CEGAR, по контрпримеру которого получить необходимый предикат и показать корректность.
